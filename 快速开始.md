## 快速开始

### clone项目
git clone https://github.com/51hhh/D-FINE_train.git

### 数据集准备
拷贝数据集到D-FINE_train/coco文件夹

若数据集格式为yolo json
拷贝D-FINE_train/benchmark/scripts/jsonTOxml.py脚本到coco
在coco文件夹执行
python jsonTOxml.py

生成xml文件夹并且重命名为annotations
### 转化训练验证、测试集
在主目录执行
python benchmark/scripts/prepare_dataset.py

生成D-FINE_train/coco/converted

### 下载预训练模型
wget https://github.com/Peterande/storage/releases/download/dfinev1.0/dfine_s_obj2coco.pth
axel https://github.com/Peterande/storage/releases/download/dfinev1.0/dfine_s_obj2coco.pth

wget 

### 开始训练
python train.py -c ../config/volleyball_s_transfer.yml --use-amp --seed 0 -t weights/dfine_s_obj2coco.pth
